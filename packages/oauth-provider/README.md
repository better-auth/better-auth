# Better Auth Provider Plugin

## Installation

```bash
# Using npm
npm install better-auth @better-auth/oauth-provider

# Using yarn
yarn add better-auth @better-auth/oauth-provider

# Using pnpm
pnpm add better-auth @better-auth/oauth-provider

# Using bun
bun add better-auth @better-auth/oauth-provider
```

## Documentation

For more information, visit the [Better Auth OAuth Provider documentation](https://better-auth.com/docs/plugins/oauth-provider).
