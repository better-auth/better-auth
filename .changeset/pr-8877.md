add `sendOldEmailVerification` option to `emailVerification` — when set to `false`, verification is sent directly to the new email address, skipping old email confirmation. Requires password or fresh session to compensate for reduced security.

also adds `afterEmailChange` hook to `changeEmail` config and `PASSWORD_OR_FRESH_SESSION_REQUIRED` error code.
