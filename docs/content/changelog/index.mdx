## v0.5.0 date=Oct 18, 2024

Multi Session Plugin, JWT & JWKS Plugin, Change Email Flow, SSR support for nuxt, CF Workers compatibility, and many more.

### Features & Changes

- Multi Session Plugin
- JWT & JWKS Plugin
- Change Email Flow

- Add support for custom id
