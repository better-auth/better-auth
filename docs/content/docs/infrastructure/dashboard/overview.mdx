## Accessing the Dashboard

Log in to the dashboard at your Better Auth Infrastructure dashboard URL. You'll need:

1. An account with dashboard access
2. Two-factor authentication (recommended)

## Dashboard Sections

### Overview

The main overview page provides a high-level view of your authentication system:

- **Key Metrics** - Total users, active sessions, signups today
- **Activity Graph** - User activity over time
- **Recent Activity** - Latest authentication events
- **Geographic Distribution** - User locations on a map
- **Activity Heatmap** - Activity patterns by time of day

### Users

Manage all users in your system:

- View and search users
- Create new users
- Edit user profiles
- Ban/unban users
- Delete users
- Manage sessions
- Impersonate users
- Send verification emails
- Reset passwords

See [User Management](/dashboard/users) for details.

### Organizations

Manage organizations and their members:

- View all organizations
- Create new organizations
- Manage members and roles
- Manage teams
- Handle invitations
- Configure SSO providers
- Set up directory sync

See [Organization Management](/dashboard/organizations) for details.

### Audit Logs

View comprehensive audit trails:

- User authentication events
- Profile changes
- Organization events
- Session activity
- Security events

See [Audit Logs](/dashboard/audit-logs) for details.

### Security

Monitor security events and threats:

- Security event timeline
- Threat breakdown by type
- Top blocked IPs
- Geographic threat distribution

See [Security Monitoring](/dashboard/security) for details.

### Settings

Configure your dashboard and project:

- Account settings
- Billing management
- Branding customization
- Email configuration
- Security settings
- Log drains
- Team members

See [Settings](/dashboard/settings) for details.

### Organization Selector

If you manage multiple organizations or projects, use the organization selector in the header to switch between them.

### User Menu

Access account settings, billing, and sign out from the user menu in the top right.

## Key Features

### Real-time Updates

Many dashboard sections update in real-time:

- Online user counts
- Recent activity feed
- Security alerts

### Search & Filtering

All list views support powerful search and filtering:

- Full-text search
- Date range filters
- Status filters
- Type filters

### Bulk Actions

Perform actions on multiple items at once:

- Bulk ban/unban users
- Bulk delete users
- Bulk revoke sessions
- Bulk send verification emails

### Data Export

Export data from most views for analysis:

- CSV export
- JSON export

## Permissions

Dashboard access is controlled by your plan and user role:

| Feature | Viewer | Editor | Admin | Owner |
|---------|--------|--------|-------|-------|
| View users | Yes | Yes | Yes | Yes |
| Edit users | - | Yes | Yes | Yes |
| Delete users | - | - | Yes | Yes |
| View settings | Yes | Yes | Yes | Yes |
| Edit settings | - | - | Yes | Yes |
| Billing | - | - | - | Yes |

Dashboard RBAC is available on Enterprise plans.

## Plan Limits

Different features are available based on your plan:

| Feature | Starter | Pro | Business | Enterprise |
|---------|---------|-----|----------|------------|
| Dashboard Seats | 1 | 3 | 5 | Unlimited |
| Audit Log Retention | 1 day | 7 days | 30 days | Unlimited |
| Security Events | - | 1k/mo | 10k/mo | Unlimited |
| SSO Management | - | - | Yes | Yes |
| Directory Sync | - | - | Yes | Yes |
| Log Drains | - | - | Yes | Yes |

## Mobile Support

The dashboard is responsive and works on mobile devices, though some features are optimized for desktop use:

- User lists and details
- Basic settings
- Activity monitoring

Complex operations like bulk actions are best performed on desktop.

## Keyboard Shortcuts

Navigate faster with keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `g` then `o` | Go to Overview |
| `g` then `u` | Go to Users |
| `g` then `r` | Go to Organizations |
| `g` then `a` | Go to Audit Logs |
| `g` then `s` | Go to Security |
| `/` | Focus search |
| `?` | Show shortcuts help |
