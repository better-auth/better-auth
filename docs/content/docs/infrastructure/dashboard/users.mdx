## Users List

Access the users list from the sidebar by clicking **Users**.

### View Options

The users list displays:

- User avatar and name
- Email address
- Sign-up date
- Last active time
- Email verification status
- Ban status
- Connected accounts

### Search & Filter

Find specific users with powerful search and filtering:

**Search:**
- Search by name
- Search by email
- Search by ID

**Filters:**
- Email verified / unverified
- Banned / active
- Has 2FA / no 2FA
- Date range (signed up, last active)

### Sorting

Sort the user list by:
- Name (A-Z, Z-A)
- Email
- Sign-up date (newest, oldest)
- Last active (recent, oldest)

## User Details

Click on any user to view their full profile:

### Profile Information

- **Basic Info** - Name, email, avatar
- **Account Status** - Email verified, banned status, 2FA enabled
- **Timestamps** - Created at, last active, email verified at
- **Connected Accounts** - OAuth providers linked to the account

### Sessions

View and manage the user's active sessions:

- Session token (partial)
- Device/browser information
- IP address
- Location
- Last activity
- Created date

**Actions:**
- Revoke individual sessions
- Revoke all sessions

### Organizations

View organizations the user belongs to:

- Organization name
- User's role
- Join date

### Activity

Recent activity for this user:

- Sign-in events
- Profile updates
- Password changes
- Session activity

## Creating Users

Create new users directly from the dashboard:

1. Click **Add User** button
2. Enter user details:
   - Email address (required)
   - Name (optional)
   - Password (optional - can send invite instead)
3. Configure options:
   - Send verification email
   - Send invite email
   - Set email as verified
4. Click **Create User**

### Invite Users

Instead of creating with a password, you can invite users:

1. Click **Invite User**
2. Enter their email address
3. Optionally add a custom message
4. The user receives an email to set up their account

## Editing Users

Update user information:

1. Navigate to the user's profile
2. Click **Edit** or the edit icon
3. Modify fields:
   - Name
   - Email address
   - Profile image URL
4. Click **Save Changes**

### Changing Email

When you change a user's email:

- The new email requires verification (by default)
- You can optionally mark it as verified immediately

## Banning Users

Ban users to prevent them from accessing your application:

### Single User Ban

1. Navigate to the user's profile
2. Click **Ban User**
3. Optionally enter a ban reason
4. Confirm the action

**When banned:**
- All active sessions are revoked
- User cannot sign in
- A `user_banned` event is logged

### Bulk Ban

Ban multiple users at once:

1. Select users using checkboxes
2. Click **Bulk Actions** → **Ban Selected**
3. Optionally enter a ban reason
4. Confirm the action

### Unbanning Users

1. Navigate to the banned user's profile
2. Click **Unban User**
3. Confirm the action

A `user_unbanned` event is logged.

## Deleting Users

Permanently remove users from your system:

### Single User Delete

1. Navigate to the user's profile
2. Click **Delete User**
3. Type the user's email to confirm
4. Click **Delete Permanently**

**Warning:** This action is irreversible. All user data, sessions, and organization memberships will be deleted.

### Bulk Delete

Delete multiple users at once:

1. Select users using checkboxes
2. Click **Bulk Actions** → **Delete Selected**
3. Confirm the action

## Session Management

### View All Sessions

See all active sessions for a user in their profile.

### Revoke Single Session

1. Find the session in the user's profile
2. Click the **Revoke** button
3. Confirm the action

### Revoke All Sessions

Force sign-out from all devices:

1. Navigate to the user's profile
2. Click **Revoke All Sessions**
3. Confirm the action

### Bulk Session Revoke

Revoke sessions for multiple users:

1. Select users using checkboxes
2. Click **Bulk Actions** → **Revoke All Sessions**
3. Confirm the action

## Impersonation

Sign in as another user to troubleshoot issues:

1. Navigate to the user's profile
2. Click **Impersonate User**
3. You're now signed in as that user
4. A banner shows you're impersonating
5. Click **Stop Impersonating** to return to your account

**Security notes:**
- Impersonation is logged as `user_impersonated`
- The impersonated session is marked as such
- Original admin session is preserved

## Two-Factor Authentication

Manage 2FA for users:

### View 2FA Status

The user profile shows:
- Whether 2FA is enabled
- 2FA method (TOTP, backup codes)
- When 2FA was enabled

### Enable 2FA (Admin)

1. Navigate to the user's profile
2. Click **Enable 2FA**
3. View the TOTP setup URI/QR code
4. User can scan with their authenticator app

### Disable 2FA

1. Navigate to the user's profile
2. Click **Disable 2FA**
3. Confirm the action

### View Backup Codes

1. Navigate to the user's profile
2. Click **View Backup Codes**
3. Codes are displayed (requires admin confirmation)

### Generate New Backup Codes

1. Navigate to the user's profile
2. Click **Generate New Backup Codes**
3. Old codes are invalidated
4. New codes are displayed

## Email Actions

### Send Verification Email

1. Navigate to the user's profile
2. Click **Send Verification Email**
3. User receives a new verification link

### Mark Email as Verified

1. Navigate to the user's profile
2. Click **Mark Email Verified**
3. The email is marked as verified without user action

### Bulk Send Verification

Send verification emails to multiple unverified users:

1. Select unverified users using checkboxes
2. Click **Bulk Actions** → **Send Verification Email**
3. Confirm the action

### Bulk Mark Verified

Mark multiple emails as verified:

1. Select users using checkboxes
2. Click **Bulk Actions** → **Mark as Verified**
3. Confirm the action

## Password Management

### Set Password

Set or reset a user's password:

1. Navigate to the user's profile
2. Click **Set Password**
3. Enter the new password
4. Confirm the action

The user can sign in with the new password immediately.

### Send Password Reset Email

1. Navigate to the user's profile
2. Click **Send Password Reset**
3. User receives a password reset link

## Connected Accounts

### View Linked Accounts

The user profile shows all OAuth providers linked:
- Provider name (Google, GitHub, etc.)
- Provider account ID
- Linked date

### Unlink Account

1. Navigate to the user's profile
2. Find the connected account
3. Click **Unlink**
4. Confirm the action

**Note:** You cannot unlink the last authentication method if the user has no password.

## User Statistics

View user analytics at the top of the Users page:

- **Total Users** - All registered users
- **Active Users** - Users active in the last 30 days
- **New Users** - Users signed up today/this week
- **Online Now** - Currently active users

## Exporting Users

Export user data for analysis:

1. Apply any desired filters
2. Click **Export**
3. Choose format (CSV, JSON)
4. Download the file

**Exported fields:**
- User ID
- Name
- Email
- Email verified status
- Created date
- Last active date
- Ban status
