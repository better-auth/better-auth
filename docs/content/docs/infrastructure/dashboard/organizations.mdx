title: Organization Management
description: The Organizations section provides tools for managing multi-tenant structures including organizations, teams, members, and invitations.
---


## Organizations List

Access organizations from the sidebar by clicking **Organizations**.

### View Options

The organizations list displays:

- Organization name and logo
- Member count
- Creation date
- Owner information

### Search & Filter

Find specific organizations:

**Search:**
- Search by name
- Search by ID
- Search by owner email

**Filters:**
- Date range (created)
- Member count range

## Organization Details

Click on any organization to view its full details:

### Overview Tab

- **Basic Info** - Name, slug, logo, metadata
- **Statistics** - Member count, team count, pending invitations
- **Timestamps** - Created at, last updated

### Members Tab

View and manage organization members:

- Member name and email
- Role (Owner, Admin, Member, or custom)
- Join date
- Teams membership

### Teams Tab

View and manage teams within the organization:

- Team name
- Member count
- Description

### Invitations Tab

View pending and past invitations:

- Invitee email
- Role offered
- Sent date
- Status (pending, accepted, expired, cancelled)
- Inviter name

### SSO Tab

Configure Single Sign-On providers (Business+ plans):

- SSO provider configuration
- Domain verification status
- Connection status

### Directory Sync Tab

Configure SCIM directory sync (Business+ plans):

- Directory connection status
- Last sync time
- Connected identity provider

## Creating Organizations

Create new organizations directly from the dashboard:

1. Click **Create Organization**
2. Enter organization details:
   - Name (required)
   - Slug (auto-generated from name, or custom)
   - Logo URL (optional)
3. Select the owner (existing user)
4. Click **Create Organization**

## Editing Organizations

Update organization information:

1. Navigate to the organization's profile
2. Click **Edit** or the settings icon
3. Modify fields:
   - Name
   - Slug
   - Logo URL
   - Metadata
4. Click **Save Changes**

## Deleting Organizations

Permanently remove an organization:

1. Navigate to the organization's profile
2. Click **Delete Organization**
3. Type the organization name to confirm
4. Click **Delete Permanently**

**Warning:** This deletes all associated data including:
- All member associations
- All teams
- All pending invitations
- SSO configurations
- Directory sync configurations

## Member Management

### Adding Members

Add existing users to an organization:

1. Navigate to the organization's Members tab
2. Click **Add Member**
3. Search for the user by email
4. Select a role
5. Click **Add Member**

### Inviting Members

Invite new users to join the organization:

1. Navigate to the organization's Members tab
2. Click **Invite Member**
3. Enter the invitee's email
4. Select a role
5. Optionally add a custom message
6. Click **Send Invitation**

The invitee receives an email with a link to accept.

### Updating Member Roles

Change a member's role:

1. Navigate to the organization's Members tab
2. Find the member
3. Click the role dropdown or **Edit**
4. Select the new role
5. Confirm the change

### Removing Members

Remove a member from the organization:

1. Navigate to the organization's Members tab
2. Find the member
3. Click **Remove** or the remove icon
4. Confirm the action

The member loses access immediately but retains their user account.

## Role Management

### Default Roles

Better Auth provides default roles:

| Role | Description |
|------|-------------|
| **owner** | Full access, can delete organization |
| **admin** | Manage members, teams, settings |
| **member** | Basic access |

### Custom Roles

If your Better Auth configuration includes custom roles, they appear in the role selector.

## Team Management

Teams allow organizing members into groups within an organization.

### Creating Teams

1. Navigate to the organization's Teams tab
2. Click **Create Team**
3. Enter team details:
   - Name (required)
   - Description (optional)
4. Click **Create Team**

### Editing Teams

1. Navigate to the team
2. Click **Edit**
3. Modify name or description
4. Click **Save Changes**

### Deleting Teams

1. Navigate to the team
2. Click **Delete Team**
3. Confirm the action

**Note:** Deleting a team removes team membership but doesn't affect user accounts or organization membership.

### Managing Team Members

**Add member to team:**
1. Navigate to the team
2. Click **Add Member**
3. Select from organization members
4. Click **Add**

**Remove member from team:**
1. Navigate to the team
2. Find the member
3. Click **Remove**
4. Confirm the action

## Invitation Management

### Viewing Invitations

The Invitations tab shows:

- **Pending** - Awaiting response
- **Accepted** - User joined via this invitation
- **Expired** - Past expiration date
- **Cancelled** - Manually cancelled

### Resending Invitations

Resend an invitation email:

1. Find the pending invitation
2. Click **Resend**
3. A new email is sent to the invitee

### Cancelling Invitations

Cancel a pending invitation:

1. Find the pending invitation
2. Click **Cancel**
3. Confirm the action

The invitation link becomes invalid.

## SSO Configuration

Configure SSO providers for the organization (Business+ plans).

### Adding an SSO Provider

1. Navigate to the organization's SSO tab
2. Click **Add SSO Provider**
3. Select provider type:
   - SAML 2.0
   - OIDC (OpenID Connect)
4. Enter configuration:
   - Provider name
   - Domain(s)
   - Metadata URL or manual configuration
5. Click **Create Provider**

### Domain Verification

Verify ownership of SSO domains:

1. Navigate to the SSO provider
2. Click **Verify Domain**
3. Add the provided DNS TXT record
4. Click **Check Verification**

Or use HTTP verification by placing a file at the specified URL.

### Testing SSO

After configuration:

1. Click **Test Connection**
2. You'll be redirected to the identity provider
3. Complete authentication
4. Verify the test results

### Editing SSO Configuration

1. Navigate to the SSO provider
2. Click **Edit**
3. Modify settings
4. Click **Save Changes**

### Removing SSO Provider

1. Navigate to the SSO provider
2. Click **Delete**
3. Confirm the action

**Warning:** Users who only sign in via SSO will need an alternative authentication method.

## Directory Sync (SCIM)

Configure automatic user provisioning (Business+ plans).

### Creating a Directory Connection

1. Navigate to the organization's Directory Sync tab
2. Click **Create Directory**
3. Enter a name for the connection
4. Copy the provided:
   - SCIM Endpoint URL
   - Bearer Token
5. Configure your identity provider with these values
6. Click **Done**

### Viewing Directory Status

The directory shows:

- Connection status
- Last sync time
- Users/groups synced
- Recent sync errors (if any)

### Regenerating Tokens

If a token is compromised:

1. Navigate to the directory connection
2. Click **Regenerate Token**
3. Update your identity provider with the new token

### Deleting a Directory

1. Navigate to the directory connection
2. Click **Delete Directory**
3. Confirm the action

**Note:** Users provisioned via SCIM remain in the organization but won't receive updates.

| `invite_rejected` | Invitation rejected |

| `team_updated` | Team settings changed |
| `team_deleted` | Team deleted |
| `team_member_added` | Member added to team |
| `team_member_removed` | Member removed from team |
