## Overage Pricing

For Pro and Business plans, overages are charged at:

| Add-on | Price |
|--------|-------|
| Additional seats | $10/month per seat |
| Event overage | $5 per 100,000 events |
| Security event overage | $100 per 100,000 events |

<Callout>
Starter plan events stop being tracked when limits are exceeded. Enterprise plans have unlimited usage.
</Callout>

## What Counts as an Event?

<Tabs items={["Tracked Events", "Not Counted"]}>
  <Tab value="Tracked Events">
    - User sign-ups and sign-ins
    - Sign-outs
    - Password changes and resets
    - Email verifications
    - Profile updates
    - Session creations
    - Organization, team, and invitation events
  </Tab>
  <Tab value="Not Counted">
    - API calls to read data
    - Dashboard views
    - Failed authentication attempts (counted as security events)
    - SCIM operations
  </Tab>
</Tabs>
