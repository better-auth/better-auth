## Overage Pricing

For Pro and Business plans, overages are charged at:

| Add-on | Price |
|--------|-------|
| Additional seats | $10/month per seat |
| Event overage | $5 per 100,000 events |
| Security event overage | $100 per 100,000 events |

<Callout>
Starter plan events stop being tracked when limits are exceeded. Enterprise plans have unlimited usage.
</Callout>

## What Counts as an Event?

<Tabs items={["Tracked Events", "Not Counted"]}>
  <Tab value="Tracked Events">
    - User sign-ups and sign-ins
    - Sign-outs
    - Password changes and resets
    - Email verifications
    - Profile updates
    - Session creations
    - Organization, team, and invitation events
  </Tab>
  <Tab value="Not Counted">
    - API calls to read data
    - Dashboard views
    - Failed authentication attempts (counted as security events)
    - SCIM operations
  </Tab>
</Tabs>

## Billing

### Payment Methods

- **All plans**: Credit/debit cards (Visa, Mastercard, Amex)
- **Enterprise**: Wire transfer available

### Billing Cycle

- Monthly billing by default
- Annual billing available with discount (contact sales)
- Invoices available in **Settings** → **Billing**

## Managing Your Plan

<Steps>
  <Step>
    ### Upgrade

    Go to **Settings** → **Billing** → **Upgrade Plan**. New features are available immediately with prorated charges.
  </Step>
  <Step>
    ### Downgrade

    Go to **Settings** → **Billing** → **Change Plan**. Takes effect at your next billing cycle.
  </Step>
  <Step>
    ### Cancel

    Go to **Settings** → **Billing** → **Cancel Subscription**. Access continues until period end, data retained for 30 days.
  </Step>
</Steps>
