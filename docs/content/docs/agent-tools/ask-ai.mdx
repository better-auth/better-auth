title: Ask AI
description: Ask questions about Better Auth from the documentation site using the in-doc AI chat.
---

The docs layout includes an **Ask AI** button (`cmd/ctrl + I`) which allows you to ask questions about the documentation. Answers are grounded in the same content you are reading on the site.

Use it when you want quick clarification without leaving the docs, or when you need a pointer to the right guide or API page.
