## Using Plugins

### Server-Side Setup

Add plugins to your Better Auth configuration:

```ts title="auth.ts"
import { betterAuth } from "better-auth"
import { admin, twoFactor, passkey } from "better-auth/plugins"

export const auth = betterAuth({
    database: {
        // your database config
    },
    plugins: [
        admin(),
        twoFactor(),
        passkey({
            rpName: "My App",
            rpID: "example.com"
        })
    ]
})
```

### Client-Side Setup

Most plugins require both server and client configuration:

```ts title="auth-client.ts"
import { createAuthClient } from "better-auth/client"
import { adminClient, twoFactorClient, passkeyClient } from "better-auth/client/plugins"

export const authClient = createAuthClient({
    baseURL: "http://localhost:3000",
    plugins: [
        adminClient(),
        twoFactorClient(),
        passkeyClient()
    ]
})
```

## CLI Command

List all available plugins using the CLI:

```bash
# List all plugins
npx @better-auth/cli list

# View detailed documentation for a specific plugin
npx @better-auth/cli list passkey

# Show verbose information for all plugins
npx @better-auth/cli list --verbose
```
